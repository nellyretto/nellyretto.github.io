Linux's *.Chronicles

What is the ls command?

Being one of the most basic commands in Linux, the function of ls is to list the directories, including its name and features. By default, it lists the name of the current directory that it’s in.

What is the function and what does ls *.c command does?

Like any other command, ls has many options to extend its function. In this case the command ls *.c serves the purpose of listing all files in the current directory that have a .c extension. Typically, it denotes C programming language source files.

Here is what each commands serves:

Ls = list the current directory.

*.c =is the pattern where * is a wildcard character that matches zero or more characters.

.c =specifies that the file name must end with .c.
